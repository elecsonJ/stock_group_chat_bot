# 8-Step 요약 (Legacy Alias)

이 문서는 유지보수 편의를 위해 축약판으로 유지됩니다.  
**최신 동작 명세는 `docs_debate_process.md`를 기준**으로 사용하세요.

## 8-Step 개요
1. 온톨로지 플랜 생성
2. RAG 주입(로컬 모델 정상 시)
3. Fact-Sheet 구성
4. 블라인드 1라운드
5. 교차 반박 라운드 + SEARCH 리서치
6. 최종 결론 태그 도출(`[최종 선택: ...]`)
7. 규칙 기반 만장일치/불합치 판정
8. 최종 판결 저장 + Evidence 저장

## 참고
1. SEARCH 근거가 없으면 결론 전 강제 리서치가 수행됩니다.
2. 로컬 모델 장애 시 강등 모드로 동작합니다.

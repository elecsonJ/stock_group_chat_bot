# 하드코어 퀀트 에이전트 워크플로우 (2026 Advanced Architecture)

이 문서는 AI 주식 투자 봇이 어떻게 16GB VRAM 로컬 환경과 프론티어 API 모델들을 융합하여 "진짜 애널리스트" 수준의 심도 깊은 리서치와 토론을 수행하는지 그 백엔드 프로세스를 설명합니다.

## 핵심 도입 기능
1. **Full-Text Scraping (원문 스크래핑)**: 단순 뉴스 스니펫(2~3줄)이 아닌 기사 원문을 통째로 긁어옵니다.
2. **Quantitative Data (정량 데이터)**: `yfinance`를 연동하여 실시간 주가, PER 등 팩트 기반의 숫자를 AI 뇌에 직접 꽂아 넣습니다.
3. **Hybrid RAG Memory (과거의 기억)**: 과거에 진행했던 토론 요약과 판결문들을 주제망(Lexical Search)을 통해 끌어와 현재 토론의 뼈대 논리로 주입합니다.
4. **CoT Token Stripping (컨닝 및 누수 방지)**: AI가 발언 전 고민하는 과정(`<thought>`)은 사용자에게만 공유되며, 봇들의 공통 대화방(History)에서는 정규식으로 깨끗하게 잘려나가(Strip), 타 모델이 남의 속마음을 훔쳐보거나 전체 토큰 한도(Token Limit)가 터지는 것을 막습니다.

## 🔄 전체 동작 프로세스 (Workflow)

### Step 1: 주제 분석 및 자율 리서치 (Plan & Search)
1. 사용자가 디스코드에 `!토론 [주제]`를 입력합니다.
2. **사전 리서치 (프론티어 AI의 능동적 요청)**
   - API 모델은 주제를 보고 자신이 모르는 최신 정보나 재무 데이터가 있다면 발언을 유보하고 아래와 같이 요청합니다.
     - `[요청:웹검색] "검색어"`
     - `[요청:주식데이터] "티커(예: NVDA)"`
3. **로컬 행동대원 (gpt-oss:20b)의 개입**
   - 파이썬 메인 코드가 요청을 가로채어 `Duckduckgo` 풀 텍스트 크롤링 또는 `yfinance` API를 호출합니다.
   - 로컬 20B 모델 수십 편의 원문을 정독하고 **[경쟁사 동향, 호재/악재, 실적 수치]**가 포함된 구조화된 리서치 리포트를 발간합니다.
   - 이 리포트가 해당 API 모델에게 피드백으로 돌아갑니다.

### Step 2: 1라운드 - 데이터 기반 초기 논거 제시 (Execute & CoT)
1. 리서치 리포트(팩트)를 장전한 API 모델들이 1라운드 주장을 펼칩니다.
2. 이때, 모델들은 무조건 **`<thought> ... </thought>`** 태그 안에 자신의 분석 논리를 자세히 작성하여 사용자에게 보여줍니다.
3. **[핵심 메커니즘] 발언이 끝날 때마다 로컬 시스템이(`<thought>`) 텍스트 덩어리를 정규식으로 삭제(Stripping)한 뒤에만 공통 `history`에 병합합니다.**
4. 이 조치 덕분에, API 모델들은 자신의 순수 주장만으로 상대 모델을 타격하게 되며 비용절감 및 독립적 사고(Anti-collusion)가 보장됩니다.

### Step 3: 2라운드 - 저격 및 즉각 방어권 발동 (Dynamic Debate)
1. 모델들은 이전 발언자들의 `<thought>` 프로세스까지 들여다보며 논리적 비약을 찾아냅니다.
2. **[ACK] 토큰 세이브**: 만약 앞 모델의 의견에 100% 동의하여 덧붙일 말이 없다면 `[ACK] 동의합니다`를 선언하고 턴을 넘깁니다. 
3. **[의지 반영: 즉각 방어권 (Emergency Defense)]** 
   - 발언자가 특정 모델을 저격하기 위해 `[조준:Claude]` 처럼 지목하여 팩트를 후벼파면, 파이썬 매니저는 정해진 발언 순서를 멈춥니다.
   - 공격받은 모델(Claude)에게 아주 짧은 `[즉각 방어의 시간]`을 부여하여, 공격에 대한 논리적 재반박을 하도록 강제합니다. 방어가 끝나면 다시 본래의 순서로 돌아갑니다.

### Step 4: 3라운드 - 로컬 수석 판사의 최종 컨센서스 (Consensus)
1. 치열한 난타전이 끝나면, 세 AI는 최종적으로 타협점(Consensus)을 도출하기 위한 마지막 의견 조율 문장을 뱉습니다.
2. **로컬 20B 투자 판사**가 전체 토론(CoT, 리서치 리포트, 공격/방어 로그 포함)을 정독합니다.
3. 세 모델의 의견이 합치되었는지, 투자 전략은 무엇인지, 실패 시 손절 라인은 어디인지 추출하여 완벽한 포맷의 JSON 리포트를 생성합니다.
4. 해당 JSON과 모든 토론 로그는 SQLite 데이터베이스에 영구 보존됩니다.
